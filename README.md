## Master Every Moment of Information Flow
**simpleCopy** transforms how you seize, collect, and reuse information with NVDA – turning fleeting spoken output, selected text, and web links into instantly accessible assets.

* **Lightning-fast URL & hyperlink capture** – grab page or link destinations with clear spoken confirmation.
* **Dynamic text accumulation** – append selections seamlessly to build notes without losing a single character.
* **Powerful speech history mastery** – copy, append, or accumulate everything NVDA speaks.

### Hot Keys

| Key | Action | Description |
|---|---|---|
| **Ctrl+Shift+A** | URL Capture | **1 Tap:** Page URL / **2 Taps:** Link URL |
| **Ctrl+Shift+C** | Text Copy | **1 Tap:** Copy/Append / **2 Taps:** Clear Clipboard |
| **F9** | Speech Power | **1 Tap:** Copy Last / **2 Taps:** Append / **3 Taps:** Copy Log / **4 Taps:** Open File |
