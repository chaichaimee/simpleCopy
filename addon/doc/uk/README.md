## Контролюйте кожен момент інформаційного потоку
**Просте копіювання** змінює підхід до збору, накопичення та повторного використання інформації за допомогою NVDA — перетворюючи швидке мовлення, вибраний текст та посилання на формат, доступний миттєво.

* **Миттєве копіювання URL та гіперпосилань** — копіюйте адреси сторінок або посилань із чітким голосовим підтвердженням.
* **Динамічне накопичення тексту** — плавно додавайте вибраний текст для створення нотаток, не втрачаючи жодного символу.
* **Потужне керування історією мовлення** — копіюйте, додавайте або зберігайте все, що озвучує NVDA.

### Гарячі клавіші

| Комбінація | Дія | Опис |
|---|---|---|
| **Ctrl+Shift+A** | Захоплення URL | **натисніть один раз:** URL сторінки / **натисніть двічі:** URL посилання |
| **Ctrl+Shift+C** | Копіювання тексту | **натисніть один раз:** Копіювати/Додати / **натисніть двічі:** Очистити буфер обміну |
| **F9** | Мовлення | **натисніть один раз:** Копіювати останнє / **натисніть двічі:** Додати / **натисніть тричі:** Копіювати журнал / **натисніть чотири рази:** Відкрити файл |
