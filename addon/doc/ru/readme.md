## Овладейте каждым моментом информационного потока
**simpleCopy** меняет способ захвата информации с NVDA, превращая мимолетную речь и ссылки в мгновенно доступные данные.

* **Молниеносный захват URL** – Копируйте адреса страниц с четким голосовым подтверждением.
* **Динамическое накопление текста** – Добавляйте фрагменты для создания заметок без потерь.
* **Мастерство истории речи** – Копируйте всё, что произносит NVDA, в реальном времени.

### Горячие клавиши

| Клавиши | Действие | Описание |
|---|---|---|
| **Ctrl+Shift+A** | Захват URL | **1 наж:** URL страницы / **2 наж:** URL ссылки |
| **Ctrl+Shift+C** | Копирование | **1 наж:** Копировать/Добавить / **2 наж:** Очистить буфер |
| **F9** | Мощь Речи | **1x:** Посл. фраза / **2x:** Добавить / **3x:** Копировать лог / **4x:** Открыть файл |
