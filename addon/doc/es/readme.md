## Domina cada momento del flujo de información
**simpleCopy** transforma la forma en que capturas, recopilas y reutilizas la información con NVDA, convirtiendo la voz fugaz en activos accesibles al instante.

* **Captura de URL ultrarrápida** – Obtén destinos de páginas con confirmación hablada clara.
* **Acumulación dinámica de texto** – Añade selecciones sin problemas para crear notas.
* **Dominio del historial de voz** – Copia o acumula todo lo que NVDA dice en tiempo real.

### Teclas Rápidas

| Tecla | Acción | Descripción |
|---|---|---|
| **Ctrl+Mayús+A** | Captura URL | **1 toque:** URL página / **2 toques:** URL enlace |
| **Ctrl+Mayús+C** | Copiar Texto | **1 toque:** Copiar/Añadir / **2 toques:** Limpiar portapapeles |
| **F9** | Poder de Voz | **1x:** Copiar última / **2x:** Añadir / **3x:** Copiar log / **4x:** Abrir archivo |
