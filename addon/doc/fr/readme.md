## Maîtrisez chaque instant du flux d'informations
**simpleCopy** transforme la façon dont vous saisissez, collectez et réutilisez les informations avec NVDA en ressources instantanément accessibles.

* **Capture d'URL ultra-rapide** – Saisissez les destinations avec une confirmation vocale claire.
* **Accumulation dynamique de texte** – Ajoutez des sélections pour créer des notes sans perte.
* **Maîtrise de l'historique vocal** – Copiez ou accumulez tout ce que NVDA prononce.

### Touches de Raccourci

| Touche | Action | Description |
|---|---|---|
| **Ctrl+Maj+A** | Capture URL | **1 appui:** URL page / **2 appuis:** URL lien |
| **Ctrl+Maj+C** | Copie Texte | **1 appui:** Copier/Ajouter / **2 appuis:** Vider presse-papiers |
| **F9** | Puissance Vocale | **1x:** Copier dernier / **2x:** Ajouter / **3x:** Copier log / **4x:** Ouvrir fichier |
