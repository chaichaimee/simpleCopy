Simple Copy


Enhanced clipboard management and data copying for NVDA


author: chai chaimee

url: https://github.com/chaichaimee/simpleCopy

Description

Simple Copy is an NVDA (NonVisual Desktop Access) add-on designed to significantly improve and extend clipboard management and data copying capabilities for screen reader users.

This add-on provides streamlined workflows for common tasks:

  • Appending selected text to existing clipboard content with a single command
  • Copying browser URLs and hyperlink addresses with intelligent browser detection
  • Utilizing an intuitive single-tap/double-tap system for accessing multiple functions from the same keystrokes

The add-on integrates seamlessly with NVDA and supports all major browsers including Chrome, Firefox, Edge, Safari, Opera, and Brave. It's designed specifically for compatibility with the NVDA 2025.x API.


Hot Keys

The add-on uses an efficient tap-based system where single and double taps of the same keystroke perform different but related functions.

Control+Shift+A - URL Management

Single Tap (press once): Copy the current browser URL to clipboard

Double Tap (press twice quickly): Copy the URL of the focused hyperlink

When pressed in an editable text field, the original keystroke is passed through to the application.


Control+Shift+C - Clipboard Management

Single Tap (press once): Copy or append selected text to clipboard

Double Tap (press twice quickly): Clear all clipboard content

If no text is selected when using the append function, the original keystroke is passed through.

Tap Timing: For a double tap to be recognized, the second press must occur within 0.5 seconds of the first press. If more time elapses, NVDA will treat them as separate single taps.


Features

1. Smart Browser URL Copying

Gesture: Control+Shift+A (Single Tap)

When active in a supported web browser, this function copies the complete URL of the current webpage to the clipboard. The add-on automatically detects when you're in a browser environment and adjusts its behavior accordingly.

Supported browsers: Chrome, Firefox, Edge, Opera, Safari, Brave

2. Hyperlink URL Extraction

Gesture: Control+Shift+A (Double Tap)

This function extracts the URL from hyperlinks using multiple methods for maximum compatibility:

  • Standard property access
  • UIA (UI Automation) fallback for modern browsers
  • IAccessible fallback for legacy support
  • Parent element searching if the focused object isn't directly a link

3. Text Append with Intelligent Formatting

Gesture: Control+Shift+C (Single Tap)

This core function provides intelligent text appending with automatic formatting:

  • Appends to existing content: If the clipboard contains text, the selected text is appended with proper line separation
  • Creates new content: If the clipboard is empty, the selected text is copied as new content
  • Format normalization: Automatically normalizes line endings to consistent format
  • Non-text protection: Detects and prevents appending to non-text clipboard content

4. Clipboard Clearing

Gesture: Control+Shift+C (Double Tap)

Quickly clear all content from the Windows clipboard with audio confirmation. This is particularly useful for security purposes or when preparing the clipboard for new content.

5. Advanced Text Selection Detection

The add-on employs sophisticated methods to retrieve selected text:

  • Primary method using treeInterceptor or focus objects
  • Fallback method simulating Ctrl+C when the primary method fails
  • Preserves original clipboard content during text extraction operations

6. Browser Detection & Context Awareness

The add-on intelligently detects the application context:

  • Automatically identifies when you're working in a web browser
  • Adjusts URL-related functions based on current context
  • Passes through original keystrokes in non-browser contexts or editable fields


Technical Note: This add-on is built specifically for the NVDA 2025.x API. It uses proper Python tools and linters to ensure code quality and follows NVDA add-on development best practices.
